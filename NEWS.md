# WeinR 0.1.0 (2026-02-24)

## Initial Release

### Features
- Quality control workflow (CreateReport)
- Read processing workflow (ProcessReads)  
- MultiQC-compatible HTML reports
- Support for FASTQ and unaligned BAM files
- High-resolution plot exports (PNG, 300 DPI)
- Comprehensive quality metrics

### Known Issues
- Large files (>5GB) may require significant memory and run faster as .gz files
